# 관리자 페이지 개선 계획

## 개선 목표
관리자 페이지 프로젝트를 취업 포트폴리오용 프로젝트로 정리한다.

## 우선 개선할 부분
- README에 프로젝트 소개와 주요 기능 보강
- 화면 캡처 이미지 추가
- 반복되는 UI 구조 정리
- 반응형 레이아웃 확인 및 수정
- 트러블슈팅 문서 작성

## 정처기 필기 기간 동안의 개발 방향
정처기 필기 준비 기간에는 큰 기능 구현보다는 문서 정리, UI 오탈자 수정, README 보강처럼 짧은 시간 안에 가능한 작업 위주로 진행한다.

## 시험 이후 진행할 작업
- 관리자 페이지 상세 설명 정리
- 포트폴리오 사이트와 배포 링크 연결
- 컴포넌트 구조 리팩토링
- 이력서용 프로젝트 설명 작성
