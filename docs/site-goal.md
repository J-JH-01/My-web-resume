# 일본어 뉘앙스 학습 사이트 목표 정리

## 프로젝트 방향

일본어 단어와 표현을 단순히 한국어 뜻만 외우는 방식이 아니라, 실제 대화에서 어떤 상황에 쓰이고 어떤 느낌을 주는지 이해할 수 있도록 돕는 학습 사이트를 만든다.

## 핵심 아이디어

- 사용자가 일본어 단어 또는 표현을 검색한다.
- 표현의 기본 뜻뿐 아니라 실제 사용 상황, 어감, 예문을 함께 제공한다.
- 비슷한 표현끼리 비교해서 차이를 이해할 수 있도록 정리한다.
- 단어를 암기하는 사이트가 아니라, 표현의 뉘앙스를 이해하는 학습 사이트를 목표로 한다.

## 우선 구현 목표

- 일본어 표현 목록 페이지
- 표현 상세 페이지
- 표현의 뜻, 뉘앙스, 예문 정리
- 비슷한 표현 비교 기능
- 즐겨찾기 또는 복습 목록 기능

## 제외할 기능

- AI 대화 기능
- AI 자동 채점 기능
- 실시간 번역 기능

오늘은 프로젝트 방향에서 AI 기능을 제외하고, 구현 가능한 일본어 뉘앙스 학습 사이트로 목표를 수정한다.
