# 일본어 뉘앙스 학습 사이트 MVP 범위

## MVP 목표

처음부터 기능을 크게 만들지 않고, 일본어 표현의 뜻, 뉘앙스, 예문을 정리해서 보여주는 기본 학습 사이트를 먼저 구현한다.

## 1차 구현 기능

- 일본어 표현 목록 페이지
- 표현 상세 페이지
- 표현의 기본 뜻 표시
- 표현의 뉘앙스 설명
- 실제 사용 예문 표시
- 비슷한 표현 비교 메모

## 제외할 기능

- AI 대화 기능
- AI 자동 채점 기능
- 실시간 번역 기능
- 복잡한 회원 시스템
- 유료 결제 기능

## 개발 방향

취업 포트폴리오용 프로젝트이므로 화려한 기능보다, 직접 구현하고 설명할 수 있는 구조를 우선한다.
처음에는 정적 데이터 또는 간단한 JSON 데이터로 시작하고, 이후 필요하면 DB 연동을 추가한다.
